Problèmes Mathématiques

🟢 Niveau Facile

Problème 1 :
Montrer que : 1 + 2 + ... + n = n(n+1)/2

Difficulté : ★☆☆☆☆

On peut associer les termes : (1+n), (2+n-1), etc. Chaque paire vaut n+1. Il y a n/2 paires → somme = n(n+1)/2.

🟡 Niveau Intermédiaire

Problème 2 :
Calculer la somme des carrés : 1² + 2² + ... + n²

Difficulté : ★★☆☆☆

La formule est : n(n+1)(2n+1)/6. Elle se démontre par récurrence ou méthodes classiques.

🔴 Niveau Avancé (IMC / Olympiades)

Problème 3 :
Trouver tous les entiers n tels que n² + n + 1 soit premier.

Difficulté : ★★★★☆

On teste plusieurs valeurs. Pour n ≥ 2, l'expression devient souvent composée. Les cas particuliers donnent les solutions. (À compléter avec une vraie démonstration détaillée)
